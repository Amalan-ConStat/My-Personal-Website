How to find your R package is simply a blog post helping people to provide a list of websites. This websites were useful for me while developing my own package fitODBODto be sure that it is a unique package and to choose what my functions should be able to do.

This is a list with 16 items

1. Google

When you have no idea to find a package first thing is to “Google”.

Link

2. CRAN

Official website to find standard packages. The packages downloaded here will have documentation manuals, vignettes and sometimes journal articles which would simplify work for people who use them.

Link

3. Bio - Conductor

This is another location to publish your R package, but only related to the field of biology.

Link

4. GitHub pages

If CRAN or Bio - Conductor is with high standards or too much work for your package we can still publish it. The ideal place is GitHub.

Link

5. Rdocumentation

This is a place to find interactive documentation for the packages in CRAN, Bio - Conductor and GitHub. Further they include functions with examples, which is very useful.

Link

6. Crantastic

All packages which are a part of CRAN is in this website. We can search packages based on Authors, package name, reviews and tags.

Link

7. rpackages

Similar to crantastic this website also provides information to CRAN packages.

Link

8. R - Opensci

Search range for R packages in this website is vary lengthy and informative. I would say better than previous ones.

Link

9. Rseek

This is like search engine for R packages.

Link

10. R Site Search

Link

11. R-forge

Projects related to R are mentioned in this website and how progress has been made on them is also here.

Link

12. AwesomeR

This is a website which has R packages based on topics related to statistics. Some of these topics are Machine Learning, Bayesian, Optimization, Bio statistics and much more.

Link

13. CRAN Task View

Topic related R packages are bundled together in this website. Further, the topics give a brief explanation, but the webpages give an extensive amount of information about what is unique is these packages.

Link

14. Rstudio - Rpackages

Several crucial packages which would be very useful are considered here. All of these packages are projects.

Link

15. stack overflow - r

If you cannot achieve something very specific related to it is possible to use the website. It provides answers from other R users, sometimes even blogs related to the issue with solutions.

Link

16. CRANalerts

This is an email service which would alert us regarding specific packages. Whenever there is an update for a chosen package we would receive an email alert.

Link
